# avcparser

## ABOUT

This to analyze AVC log of SEAndroid

## USAGE

```bash
avcparser [logfile...]
```

Read manual for more info:

```bash
man 1 avcparser
```

## INSTALLATION

```shell
git clone https://github.com/Arondight/avcparser.git
cd avcparser
perl Makefile.PL
make
sudo make install
```
