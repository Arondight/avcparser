# NAME

avcparser - Analyze AVC log of SEAndroid

# SYNOPSIS

```shell
avcparser [logfile...]
```

Read manual for more info:

```shell
man 1 avcparser
```

# BUILD

```shell
git clone https://github.com/Arondight/avcparser.git avcparser
cd avcparser
perl Makefile.PL
make
sudo make install
```
